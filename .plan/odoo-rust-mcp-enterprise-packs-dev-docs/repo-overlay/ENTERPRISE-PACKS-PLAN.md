# Enterprise Packs Plan

Canonical implementation documentation:

```text
docs/src/developer/enterprise-packs/
```

Recommended first release scope:

1. protocol and registry correctness;
2. policy engine in audit mode;
3. capability snapshots;
4. instance-scoped exposure;
5. Manufacturing read and planning pack;
6. Spreadsheet/Dashboard read and snapshot pack;
7. Website draft pack;
8. POS reporting pack;
9. Employee public directory pack.

Commands with stock, accounting, publication, POS cash, or employee-access
impact remain blocked until confirmation tokens and state reconciliation are
complete.
